##Linked List Manipulator
Was tested on Linux  5.4.0-56-generic 18.04.1-Ubuntu

##How to install 
1. Clone this repository.
```bash
git clone https://github.com/vomelnit/Linked_List_Manipulator
```
2. Download gtest:
```bash
sudo apt-get install libgtest-dev
```
3. Build program
```bash
make
```
3. Execute program 
```bash
./bin/program
```
